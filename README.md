Mini Healthcare Support Web App
Overview
This is a concept-level web app designed to support healthcare initiatives by providing patient support, volunteer registration, and contact forms. It aims to simplify communication and data collection for NGOs or healthcare providers.
Features
Patient Support Form: Collects basic patient information and requirements.
Volunteer Registration Form: Enables volunteers to register for NGO activities.
Contact Form: For general inquiries.
AI/Automation Idea: Chatbot concept for answering frequent questions (FAQs) and providing guidance to users.

Tech Stack
Frontend: HTML, CSS, JavaScript
Hosting: Vercel (live hosted)
AI/Automation Idea
Chatbot Concept: Provides instant answers to common queries like available services, contact info, or registration guidance.
Could be extended to auto-respond to forms or summarize collected data for NGO use.

NGO Use-Case
Helps NGOs manage patient support requests efficiently.
Simplifies volunteer registration.
Reduces response time through AI-driven guidance.
